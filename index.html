TableFactory by kstrauser

Easily create HTML, spreadsheet, or PDF tables from common Python data sources

TableFactory is a simple API for creating tables in popular formats. It acts as a wrapper around other popular Python report generators and handles all the tedious, boilerplate problems of extracting columns from input data, creating the layout, applying formatting to cells, etc.

Dependencies

xlwt (http://pypi.python.org/pypi/xlwt) for generating Excel spreadsheets ReportLab (http://www.reportlab.com/software/opensource/) for generating PDFsxlwt (http://pypi.python.org/pypi/xlwt) for generating Excel spreadsheets ReportLab (http://www.reportlab.com/software/opensource/) for generating PDFs

Install

Coming soon: PyPI, easy_install, pip, etc. Those aren't ready yet, though. To get started now, download TableFactory.py from https://github.com/kstrauser/tablefactory/blob/master/TableFactory.py and copy it somewhere in your Python path. For example, /usr/local/lib/python2.6/site-packages might be a reasonable location on a Ubuntu system.
